# thermal-storage
A program which simulates the behavior of a thermal storage

The directory has a following structure:
- data
  - properties.txt
- model
  - continuous-model
    - package
      - export.py
      - initialize.py
      - mapping.py
    - arrays.py
    - storage.py
  - single-phase-model
    - package
      - export.py
      - initialize.py
      - mapping.py
    - arrays.py
    - storage.py
  - properties.py
